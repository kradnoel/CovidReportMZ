CovidReportMZ API

CovidReportMZ API is the API for the CovidReportMZ that help people get realiable information about Covid-19 on Mozambique. The data come from the Mozambique Health Institute website (https://covid19.ins.gov.mz) using web scraping technique. The project was build for the Twilio Hackaton on DEV.

Statistics

GET https://covidreportmz-api.herokuapp.com/stats HTTP/1.1 HTTP/1.1 200 OK Content-Type: application/json { "status": "sucess", "code": 200, "data": { "date": "Actualizado 14h00 – 19/04/2020", "stats": [ { "title": "ENTRADOS", "total": "473885" } ... ] } }
